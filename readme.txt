
Operating System : Windows
Language written : Python
Python Version : Python 3.7.4
No external libraries are used.
Single file containg 4 classes : node, ring_topology, dynamic ring_topology, cov_topology

TMAN.py can be runned as : python TMAN.py N k topology

For example:
Ring Topology ==> python TMAN.py 1000 40 R

Dynamic Ring Topology ==> python TMAN.py 100 3 D 4 2 4 6 8

Cov-2 Topology ==> python TMAN.py 1000 15 C

Note: For Dyanmic ring topology, radius values are separated by space as given in above example.
